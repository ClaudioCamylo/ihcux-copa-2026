## Proposta do Projeto
O aplicativo CopaNaMão foi pensado para melhorar a experiência do torcedor dentro do estádio durante jogos da Copa do Mundo, reduzindo dificuldades comuns como filas para alimentação, perda de tempo ao buscar serviços e perda de lances importantes.

## Problema Focado
A principal dor priorizada neste protótipo foi a dificuldade do torcedor em pedir alimentos sem enfrentar filas e sem perder momentos importantes da partida.

## Justificativa de Design
O layout foi organizado priorizando rapidez de acesso e leitura clara em um ambiente barulhento e com distrações. Por isso:

- Botões grandes para funções críticas.
- Home centralizando ações principais.
- Fluxo curto para pedido de comida.
- Navegação simples para mapas e replays.
- Informações prioritárias visíveis primeiro.

## Descrição das Telas

### Home / Dashboard
Exibe placar ao vivo e atalhos rápidos para:
- mapa do estádio
- pedir comida
- replays
- estatísticas

### Mapa
Permite localizar:
- assento
- portão
- banheiros
- quiosques

### Cardápio
Lista alimentos e bebidas disponíveis para pedido.

### Carrinho
Resumo do pedido e confirmação da retirada.

### Replays
Acesso a lances recentes por replay.

### Ingresso
Exibe QR Code do ingresso e dados do setor.

## Fluxo do Usuário (Pedido de Lanche)
1. Usuário acessa Home.
2. Seleciona “Pedir Comida”.
3. Escolhe item no Cardápio.
4. Adiciona ao Carrinho.
5. Confirma pedido.
6. Recebe aviso para retirada.
